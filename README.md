A modern Task Manager built with React + TypeScript, focused on clean architecture, accessibility, and testability.
The app supports full task lifecycle management with persistent storage and a smooth user experience.

- CRUD Tasks
- Filters (All / Pending / Completed)
- Edit task inline
- Persistent storage
- Accessible UI
- Unit tested

## Tech Stack
- React
- TypeScript
- Vite
- Context + Reducer
- Vitest + RTL
